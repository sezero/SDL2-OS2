# SDL2-OS2
SDL2 libraries for OS/2, ArcaOS and eComStation.

COMPILE TOOLS
===============
* Open Watcom 1.9

    may or may not be a fault of this port.

  - https://efbe.musca.uberspace.de/

  - http://open-watcom.github.io/open-watcom/
  - https://github.com/open-watcom/
